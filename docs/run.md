# oceanica-pescados 🐟

1. Clone o repositório
```bash
git clone https://github.com/lucaspanzera1/oceanica-pescados
cd osphone
```

2. Instale as dependências
```bash
pnpm install
```

3. Configure as variáveis de ambiente
```bash
cp .env.example .env
# Configure suas variáveis de banco de dados
```

5. Inicie o servidor
```bash
pnpm run dev
```
